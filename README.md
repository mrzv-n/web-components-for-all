# Web Components for all

Демо-проект билиотеки комопонентов на Lit для тестирования совместимости с React, Vue, Angular.
Для сборки проекта и демо проекта используется Nx.

**lit** - v3.1 \
**React** - v18.3 \
**Vue** - v3.3 \
**Angular** - v18 \
**nx** - v19.3 \


## Для запуска демо
```
# Запуск демо Vue
nx serve vue

# Запуск демо Angular
nx serve angular

# Запуск демо React
nx serve react
```
